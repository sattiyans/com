## Overview

3rd SPACE by Triterra is a community-focused website for The MET Corporate Towers. It was built to present place information, event updates, and community content in one accessible site.

## Features

- Structured navigation for Home, The MET, Directory, and Blog.
- CMS-driven content areas for events, updates, and community stories.
- Mobile-responsive layout with consistent component styling.
- Clear visual hierarchy for faster content discovery.

## Future Improvements

- Add richer directory filters and category landing pages.
- Add event RSVP/lead capture integration.
- Improve content performance metrics and search indexing.
