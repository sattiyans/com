## Overview

TNB SMD is an enterprise dashboard for subsidiary governance and performance tracking. The work focused on UI modernization from Figma plus backend/controller alignment so redesigned views remained functional and production-ready.

## Features

- End-to-end dashboard UI redesign across subsidiaries, ESG, risk, HSE, and oversight modules.
- Controller and Blade integration updates to support new layouts and components.
- Role-based access + auditing preserved during redesign.
- Improved data-entry pages, filters, and dashboard readability.
- Export/report workflows retained while modernizing interface behavior.

## Future Improvements

- Add component-level performance monitoring for heavy dashboards.
- Introduce more interactive drill-down analytics.
- Expand design system coverage for faster future module rollout.
